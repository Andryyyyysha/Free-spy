# Free Spy

Бесплатный инструмент с открытым исходным кодом для логирования измененных и удаленных сообщений в Telegram через меню «Автоматизация чатов».

Проект разработан на базе библиотеки aiogram 3 и предназначен для развертывания личного экземпляра бота.

[![Deploy to Render](https://render.com/images/deploy-to-render-button.svg)](https://render.com/deploy?repo=https://github.com/Claxy-mod/Free-spy)

## Возможности

* **Поддержка любых типов сообщений** — логирует текст, фотографии, видео, голосовые сообщения, видеосообщения (кружки), документы, аудио, стикеры и GIF.
* **Логирование в реальном времени** — отслеживает события изменения и удаления сообщений в привязанных бизнес-аккаунтах.
* **Устойчивость к сбоям (буферизация)** — накапливает события во время отключения бота или отсутствия сети и присылает их единым отчетом при запуске, позволяя скачивать пропущенные медиафайлы по кнопке.
* **Динамическая маршрутизация** — бот работает в мультипользовательском режиме. Любой пользователь может привязать его через автоматизацию чатов в настройках Telegram и получать логи только в свой чат.
* **Шифрование данных (AES-128)** — сообщения шифруются ключом `ENCRYPTION_KEY` по стандарту Fernet перед записью в базу данных. Данные защищены даже при полном доступе к СУБД.

## Быстрый старт

1. Нажмите на кнопку **Deploy to Render** выше.
2. Введите `BOT_TOKEN` (токен от @BotFather) и `USER_ID` (ваш числовой ID в Telegram).
3. Дождитесь запуска и подключите бота в Telegram (**Изменить профиль -> Автоматизация чатов -> Добавить бота**).

Подробная пошаговая инструкция по ручной настройке базы данных Supabase, подключению пинга для работы 24/7 и локальному запуску находится в файле [INSTALL.md](INSTALL.md).
