# Free Spy: Бесплатная OpenSource замена Dialog Spy Bot

**Free Spy** — это современный Telegram-бот на базе библиотеки `aiogram 3`, предназначенный для перехвата, сохранения и логирования удаленных и измененных сообщений в ваших бизнес-чатах Telegram.

## Возможности
* 📁 **Поддержка любых типов сообщений**: бот логирует не только текст, но и фото, видео, голосовые сообщения (ГС), видео-сообщения (кружки), файлы, аудио, стикеры и GIF.
* 🗑 **Отслеживание удалений и изменений**: при удалении или изменении сообщения бот мгновенно отправляет уведомление владельцу с указанием автора.
* 🎥 **Авто-сохранение исчезающих медиа (В разработке / Тестировании)**: бот пытается перехватить и сохранить самоудаляющиеся («view-once») фото, видео и голосовые сообщения. Обратите внимание, что из-за ограничений безопасности Telegram API данная функция находится на стадии активного тестирования и может работать нестабильно.
* 👥 **Мультипользовательская архитектура**: бот автоматически перенаправляет уведомления именно тому пользователю, который подключил его к своему бизнес-аккаунту.

## Требования
* Python версии 3.9 и выше.

## Быстрый старт и установка

### 1. Клонирование и установка зависимостей
Склонируйте репозиторий с проектом, перейдите в него и установите необходимые зависимости:

```bash
# Создание виртуального окружения
python3 -m venv venv

# Активация виртуального окружения
source venv/bin/activate  # Для macOS/Linux
# venv\Scripts\activate  # Для Windows

# Установка библиотек
pip install -r requirements.txt
```

### 2. Настройка конфигурации
Переименуйте файл настроек-шаблонов `config.ini.example` в `config.ini`:

```bash
cp config.ini.example config.ini
```

Откройте `config.ini` текстовым редактором и заполните ваши данные:
```ini
[settings]
TOKEN = YOUR_BOT_TOKEN_HERE            # Токен вашего бота от @BotFather
USER_ID = YOUR_TELEGRAM_USER_ID_HERE   # Ваш личный ID в Telegram (для резервных уведомлений)
TIMEZONE_NAME = Europe/Moscow          # Ваш часовой пояс для форматирования времени
```

### 3. Запуск бота
Запустите скрипт `main.py`:

```bash
python main.py
```

Для запуска бота в фоновом режиме (на сервере):
```bash
nohup python3 main.py > bot.log 2>&1 &
```

## Как подключить к Telegram (Автоматизация чатов)

1. Откройте **Telegram** на телефоне и перейдите в ваш **Профиль**.
2. В правом верхнем углу нажмите **Изменить** (Edit).
3. В списке найдите раздел **Автоматизация чатов** (Chat Automation) или **Чат-боты** (Chatbots).
4. Нажмите **Добавить бота** (Add bot) и укажите юзернейм вашего бота.
5. Настройте права доступа (выберите чаты, в которых бот сможет работать).
6. Сохраните изменения.

Как только бот подключится, в консоли отобразится событие:
`Business connection established...`

Теперь бот автоматически начнет сохранять историю сообщений в локальную базу данных SQLite (`messages.db`). При удалении сообщений собеседником бот отправит вам их копии!
